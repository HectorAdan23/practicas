gestorInventario es un ejemplo de un programa que lleva el conteo de los articulos que hay en un establecimiento. Su principal uso

1)¿Qué hace?
Ya tiene un inventario ficticio, pero el objetivo del programa es agregar nuevos espacios para nuevos productos que se le
ingresen, ademas de la cantidad y precio. Tambien puede dar el valor en bolsa o total del dinero en mercancia existente, mas la que el usuario desee agregar.

2)¿Problemas?
Ninguno como la fuga de memoria o no poder manejar valores que no espera resivir

3)¿Puntos a favor?
Este es un ejemplo practico de lo que sucede dentro de un programa software que lleva el inventariado de una tienda comercial.

4)¿ejecucion? hacer descarga del archivo "gestorInventario.c", hacer "gcc gestorInventario.c -o mi ejecutable" y por ultimo arrancar el ejecutable.
